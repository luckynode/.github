### 𝗕𝗮𝗰𝗸𝗴𝗿𝗼𝘂𝗻𝗱
- 친구들의 생일을 일일이 기억하기 어려운 상황
- 선물 고르는 시간을 단축하고 싶은 상황
- 생일에 간단한 편지라도 전달하고 싶은 상황
- 생일 선물로 고가의 선물을 원함

### 𝗚𝗼𝗮𝗹𝘀
- 사용자가 자신의 생일을 친구들과 쉽게 공유
- 편지 아카이빙이 가능하며
- 위시리스트에 본인이 원하는 선물을 등록해 놓으면
- 여러 사람들이 생일 선물 펀딩을 함
- 생일인 사람과 축하하는 사람 모두를 만족할 수 있는 서비스
